Velkommen til Kapologikk

Dette er et verktøy som bruker Bayes’ teorem til å hjelpe deg med å evaluere ulike påstander og argumenter basert på evidens.

Velg hvilken valgomat du ønsker å bruke:

  • Question: If papacy is true, Catholicism is true - if not, it is not.
  • Valgomat: If Sola Scriptura is true, Protestantism is true - if not, it is not.
